# NIST CSF Gap Analysis

This project evaluates a fictional organization's cybersecurity maturity using the five NIST CSF functions. It includes a report and workbook with maturity scoring, target-state analysis, and dashboard views.

## Files
- `NIST_CSF_Gap_Analysis_Report.docx`
- `NIST_CSF_Gap_Analysis_Report.pdf`
- `NIST_CSF_Gap_Analysis_Workbook.xlsx`

## What this demonstrates
- Framework-based compliance assessment
- Maturity scoring and gap analysis
- Remediation roadmap development
- Practical dashboarding for leadership review
