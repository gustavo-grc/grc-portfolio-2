# Incident Response Tabletop Exercise

This project simulates a ransomware tabletop exercise with an action log, response roles, decision points, and a structured after-action improvement plan.

## Files
- `Ransomware_Tabletop_Exercise.docx`
- `Ransomware_Tabletop_Exercise.pdf`
- `Incident_Timeline_and_Action_Log.xlsx`

## What this demonstrates
- Incident response planning
- Cross-functional coordination
- Crisis decision-making
- Improvement tracking
