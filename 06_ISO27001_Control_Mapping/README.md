# ISO 27001 Control Mapping

This project demonstrates how selected ISO 27001 control themes can be translated into practical implementation evidence and mapped to NIST CSF references.

## Files
- `ISO27001_Control_Mapping_Report.docx`
- `ISO27001_Control_Mapping_Report.pdf`
- `ISO27001_Control_Mapping_Workbook.xlsx`

## What this demonstrates
- Framework mapping
- Audit-readiness interpretation
- Evidence-oriented control design
- Cross-framework communication
