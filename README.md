# NIST Cybersecurity Risk Assessment

This project simulates a professional cybersecurity risk assessment deliverable for a healthcare SaaS company. The package includes an executive-style report, a structured risk register, scoring logic, and a dashboard.

## Files
- `HealthData_Cloud_Risk_Assessment_Report.docx`
- `HealthData_Cloud_Risk_Assessment_Report.pdf`
- `Risk_Register.xlsx`

## What this demonstrates
- Asset and threat identification
- Qualitative risk scoring
- Risk treatment planning
- Executive reporting and audit-style documentation
